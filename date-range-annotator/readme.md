# Date Range Annotator

This is a docker image wrapper around the [Date Range Annotator](https://github.com/ewoij/date-range-annotator).

The docker image is following the component OpenMinTeD guidelines.

## Usage

```
docker run <image-name> annotates --help
```

```
usage: annotates [-h] --input INPUT_FOLDER --output OUTPUT_FOLDER

optional arguments:
  -h, --help            show this help message and exit
  --input INPUT_FOLDER
  --output OUTPUT_FOLDER
```
